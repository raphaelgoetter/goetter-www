À propos

Hello ! Je suis Raphaël, je conçois des sites internet au sein de l'agence web Alsacreations.fr, spécialisée dans l'accessibilité numérique et que j'ai co-fondée.
Je passe une grande partie de mon temps en tant qu'expert, consultant et formateur en langages web ( HTML , CSS, Responsive Webdesign, Flexbox et Grid Layout) auprès de clients nationaux et internationaux. Si ça ne vous évoque rien, dites-vous que je ne fais pas que réparer des imprimantes…
J'ai également créé Alsacreations.com, communauté d'apprentissage libre et gratuite avec forums de discussion, tutoriels et service d'emploi.

Je dispose par ailleurs de superpouvoirs indispensables dans le monde du Web : je sais marcher sur les mains (eh ouais), je sais réussir à faire des photos floues, et dans ma jeunesse je savais parler à l’oreille d’Internet Explorer 6 (avec un sonotone).

Conférences

Mon métier m'offre parfois l'occasion d'être orateur et de présenter mes passions aux cours de conférences web organisées dans divers coins de l'Hexagone, voire plus loin encore.

Parmi les conférences qui m'ont le plus marqué, j'ai d'excellents souvenirs de : Paris Web (Paris), DevFest (Nantes), Confoo (Montréal), BlendWebMix (Lyon), Web À Québec (Québec), Microsoft TechDays (Paris), la KiwiPartÿ (Strasbourg).

J'adore partager mes connaissances lorsqu'il s'agit de nouveautés CSS, de Responsive Webdesign, de Grid Layout, de performances web ou d'accessibilité numérique. N'hésitez pas à me contacter pour titiller ma curiosité !

Formations web

Qu'il s'agisse de sessions de formations chez Alsacréations à Strasbourg, ou partout en France au sein des entreprises pour former des équipes, j'interviens sur divers thèmes de compétences et plus particulièrement sur deux formations :

  • Formation Bonnes pratiques CSS3 Toutes les techniques CSS modernes et professionnelles pour faciliter vos intégrations. Bonnes pratiques et méthodologies sont au programme !
  • Formation Flexbox et Grid Layout Les mises en page Responsive n'auront plus aucun secret pour vous grâce aux nouveaux modes de positionnement CSS3 révolutionnaires.

Ces deux programmes de formation peuvent être dispensés dans vos locaux, pour une durée de 3 jours jusqu'à une semaine, et être totalement sur mesure et adaptés à vos besoins internes.

Mon petit univers

Alsacréations et autres projets web

J'ai créé le site Alsacreations.com en 2003, communauté libre d'apprentissage web dont le trafic avoisine encore aujourd'hui les 30000 visites uniques quotidiennes et les 50000 membres. Tous les contenus et tutoriels d'Alsacréations sont libres sous licence Creative Commons.

Je suis co-fondateur en 2006 et co-gérant de l'agence de conception web strasbourgeoise Alsacreations.fr.

J'organise la KiwiParty, événement gratuit organisé par Alsacréations réunissant conférences et ateliers, autour des thèmes de la conception web et de ses bonnes pratiques.

Il y a toujours un projet web libre et gratuit quelque part auquel je participe :

  • Bretzel : un outil d'assistance à la méthodologie Cube CSS conçu par l'agence web Alsacréations pour ses projets d'intégration
  • Schnaps.it : générateur de Lorem Ipsum alsacien
  • Mydevice.io : profil, valeurs et capacités des terminaux mobiles
  • KNACSS.com : mini framework CSS responsive (obsolète)

Mon bac à sable

Codepen est mon terrain de jeu privilégié pour mes expériences web de toutes sortes : je suis sans cesse en train d'y tester de nouvelles fonctionnalités CSS prometteuses.

N'hésitez pas à y faire un tour, vous inspirer, échanger avec moi, améliorer mes brouillons :

  • Collection "Grid Layout" : une liste de cas d'usage remarquables de CSS Grid Layout
  • Collection "Jack in the Flexbox" : plein d'exemple d'astuces fondées sur CSS Flexbox
  • Collection "Animations" : mes meilleures expériences en animation CSS (scroll-timeline, view-timeline, le long de tracés, etc.)
  • Collection "Container Queries" : mes tests du futur remplaçant des Media Queries
  • Collection "Typo" : tous mes exemples de fonctionnalités CSS liées à la typographie
  • Collection "Tricks" : tout ce qui mérite d'être conservé mais qui ne rentre dans aucune catégorie distincte
